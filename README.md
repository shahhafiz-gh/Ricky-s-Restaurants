# Ricky's Restaurant

A luxury dining restaurant website built with Next.js 14, React, Tailwind CSS, and Shadcn UI components.

## Getting Started

### Prerequisites

- Node.js (version 18 or higher recommended)
- npm, yarn, pnpm, or bun

### Installation & Running Locally

1. Clone the repository:
   ```bash
   git clone git@github.com:shahhafiz-gh/Ricky-s-Restaurants.git
   cd Ricky-s-Restaurants
   ```

2. Install the required dependencies:
   ```bash
   npm install
   ```

3. Start the local development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the result. You can start editing the page by modifying `src/app/page.tsx`. The page auto-updates as you edit the file.

## Features

- **Next.js 14 App Router:** Utilizing the latest Next.js routing and server components architecture.
- **Tailwind CSS & Shadcn UI:** Beautifully crafted, responsive user interfaces and components with a custom warm brown/gold luxury theme.
- **Framer Motion Elements:** Smooth page transitions, scroll-triggered reveal animations, and interactive carousels.
- **Zod Validation:** Fully type-safe and validated forms.
- **Statically Exportable:** Configured for seamless deployment onto static hosting platforms like Surge.

## Building for Production

To create an optimized production build:

```bash
npm run build
```

This will run the build script and export a fully static artifact to the `out/` directory, which can be deployed to any static site hosting service.
